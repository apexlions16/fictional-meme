# Değişiklik Günlüğü

Bu projedeki önemli değişiklikler bu dosyada belgelenir. Sürümler Semantic Versioning yaklaşımını izler.

## [1.0.0] - 2026-07-11

### Eklendi

- React 19 ve TypeScript tabanlı Fictional Meme Studio.
- Procedural şablon kütüphanesi ve kategoriye duyarlı metin motoru.
- Canvas tabanlı gelişmiş düzenleyici ve yüksek çözünürlüklü PNG çıktısı.
- Favoriler, proje kütüphanesi, JSON yedekleme ve yerel analitik.
- PWA, koyu/açık tema ve GitHub Pages dağıtımı.
- Otomatik GitHub Release üretimi ve doğrulama özetleri.
- GitHub Container Registry için çok mimarili OCI paketi.
